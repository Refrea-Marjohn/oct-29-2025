🔧 HOW TO USE
═══════════════════════════════════════════════════════════════════

FOR ADMIN:
1. Navigate to: Admin Dashboard → Document Generation
2. Select document type from dropdown
3. Fill in the required fields
4. Click "Generate Document"
5. PDF will be generated and ready to download

FOR ATTORNEY:
1. Navigate to: Attorney Dashboard → Document Generation
2. Choose client and document type
3. Complete the form
4. Generate and download PDF

FOR EMPLOYEE:
1. Navigate to: Employee Dashboard → Document Generation
2. Select appropriate document
3. Fill in all fields
4. Generate document
5. Send to client if needed
